**Amazon E-Commerce Manual Testing**



**Description**

Manual testing of Amazon web application focusing on real-time shopping flows.



**Modules Tested**

1. Product Search

2\. Product Details

3\. Cart

4\. Checkout

5\. Payment

6\. Order Tackingg



**Testing Types**

1. Functional Testing

2\. Regression Testing

3\. Negative Testing



**Tools i used**

1. Excel

2\. Jira



**Testing Approach**

Test cases were designed based on real user scenarios. Each test case was executed manually, and the actual results were compared with the expected results. Any deviation was logged as a defect in Jira with proper severity and priority.



**Outcome**

All major shopping functionalities were tested successfully. Critical and medium-severity issues related to payment validation and cart behavior were identified and documented in Jira.
